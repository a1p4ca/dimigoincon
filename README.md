# DIMIGOINcon

**Join in. Link in. DIMIGOIN.** DIMIGOINcon(이하 디미고인콘)은 한국디지털미디어고등학교 인트라넷 서비스 [DIMIGOIN](http://dimigo.in)에 사용되는 아이콘 팩입니다. Team DIMIGOIN은 아이콘을 SVG로 제작하여 [Fontello](http://fontello.com/)을 통해 웹 폰트로 완성하였습니다. 유용성과 유려함이 공존하는 디미고인콘을 만나보세요.



### 구성

더 자연스러운 흐름, 더 나은 경험을 위해 DIMIGOIN은 용도가 다른 두 종류의 아이콘이 사용되었습니다. 세세한 부분에서 자연스러움을 위해 배치된 32px 이하의 작은 아이콘과 메인 페이지에서 멋진 경험을 위해 삽입된 96px 아이콘이 있죠.

### 사용

위의 목록에 수록된 모든 아이콘은 웹 폰트를 통해 간편하게 사용할 수 있습니다.

```html

```
